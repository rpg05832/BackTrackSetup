# BackTrack

כלי קטן וקל ל‑Windows לביטול סגירות בטעות (כמו Ctrl+Shift+T בדפדפן — אבל לכל המחשב), עם לוח העתקה, קבוצות עבודה, סידור חלונות ואיחוד חלונות ללשוניות.

> A lightweight Windows tool to undo accidental closes (folders, files, apps, browser tabs), with clipboard history, workspace groups, window snapping/tiling, and merging windows into tabs.

## יכולות עיקריות

- **שחזור מהיר** של תיקיות, קבצים, אפליקציות וכרטיסיות שנסגרו — קיצור גלובלי `Ctrl + Alt + Z`.
- **לוח העתקה** ששומר את כל הטקסטים שהועתקו ומאפשר להעתיק מחדש.
- **קבוצות עבודה** — "לצלם" תיקיות + תוכנות + מסמכים פתוחים ולשמור כקבוצה, לפתיחה יחד (כחלונות נפרדים או כלשוניות).
- **סידור חלונות** — הצמדה זה לצד זה / רשת / עמודות / ראשי+צד.
- **איחוד חלונות ללשוניות** — מיזוג חלונות לחלון אחד עם טאבים (גרירת חלון על חלון, כפתור +, גרירת לשונית החוצה).
- **חלון ניהול מובנה** (WebView2), **מצב פרטי**, ועלייה אוטומטית עם המחשב.

## דרישות מערכת

- Windows 10 / 11
- ‎.NET 10 Desktop Runtime (אם חסר — Windows יציע להוריד בהרצה הראשונה)
- רכיב **WebView2** לחלון הניהול (מותקן כברירת מחדל ברוב המחשבים; אם חסר — תוצע הורדה, ובינתיים ייפתח בדפדפן)

## בנייה מהמקור

```bash
dotnet publish -c Release -r win-x64 --self-contained false -p:PublishSingleFile=true
```

הפלט הוא קובץ `BackTrack.exe` יחיד (Portable, ~1.5MB).

## הורדות

קבצי ההרצה (גרסה ניידת + תוכנת התקנה) זמינים תחת **Releases**.
